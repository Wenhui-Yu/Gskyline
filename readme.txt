This project is code for Fast Algorithms for Pareto Optimal Group-based Skyline (https://dl.acm.org/citation.cfm?doid=3132847.3132950)
All datasets used in this paper is available at: https://pan.baidu.com/s/1rHNg2biDS9gs9WeSSlV7Ww
There are three parts in the project, MSL, G_skyline, and G_clustering. In each part, execute  **_main**  file to run the code.
